# options-calculator

How many longs and shorts?

## Guide

### 0. Create .env file

```sh
touch .env
```

### 1. Fill up environmental variables

-   CONSUMER_KEY
-   REFRESH_TOKEN
-   ACCOUNT_ID
-   PADLOCAL_TOKEN

### 2. Create ./cache folder

```sh
mkdir cache
```

### 3. Run!

```sh
node app.js
```
